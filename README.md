#### WARDRIVER

This project is meant to read local bluetooth and wifi antennas while storing surrounding metadata
including GPS, tilt, compass, etc.

##### Structure

```
wardriver 
+-- src // VHDL 
+-- scripts // TCL 
+-- simulation // Verilog
|   +-- main.v
|   +-- gps.v
|   +-- bluetooth.v
|   +-- network.v
|   +-- compass.v
|   +-- tilt.v
+-- README.md
```
